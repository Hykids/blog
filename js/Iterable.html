Iterable вҖӢ

еҸҜиҝӯд»ЈеҜ№иұЎпјҲIterableпјүжҳҜжҢҮе®һзҺ°дәҶ Symbol.iterator ж–№жі•зҡ„еҜ№иұЎгҖӮеңЁ JavaScript дёӯпјҢеҸҜиҝӯд»ЈеҜ№иұЎеёёи§Ғзҡ„жңүж•°з»„гҖҒеӯ—з¬ҰдёІгҖҒMapгҖҒSet зӯүгҖӮ

Symbol.iterator ж–№жі•жҳҜдёҖдёӘж— еҸӮж•°еҮҪж•°пјҢиҝ”еӣһдёҖдёӘиҝӯд»ЈеҷЁеҜ№иұЎпјҲIteratorпјүгҖӮиҝӯд»ЈеҷЁеҜ№иұЎжҳҜдёҖдёӘеҢ…еҗ« next() ж–№жі•зҡ„еҜ№иұЎпјҢжҜҸж¬Ўи°ғз”Ё next() ж–№жі•пјҢиҝӯд»ЈеҷЁе°Ҷиҝ”еӣһдёҖдёӘеҢ…еҗ« value е’Ң done дёӨдёӘеұһжҖ§зҡ„еҜ№иұЎгҖӮе…¶дёӯпјҢvalue иЎЁзӨәеҪ“еүҚиҝӯд»Јзҡ„еҖјпјҢdone иЎЁзӨәжҳҜеҗҰиҝӯд»Је®ҢжҲҗгҖӮ

дҫӢеҰӮпјҢд»ҘдёӢжҳҜдёҖдёӘиҮӘе®ҡд№үиҝӯд»ЈеҷЁзҡ„дҫӢеӯҗпјҡ

js
javascriptCopy codeconst myIterable = {
  *[Symbol.iterator]() {
    yield 1;
    yield 2;
    yield 3;
  }
};

for (let value of myIterable) {
  console.log(value);
}
// иҫ“еҮәпјҡ
// 1
// 2
// 3

еңЁдёҠйқўзҡ„дҫӢеӯҗдёӯпјҢжҲ‘д»¬е®ҡд№үдәҶдёҖдёӘеҗҚдёә myIterable зҡ„еҜ№иұЎпјҢе№¶е®һзҺ°дәҶ Symbol.iterator ж–№жі•пјҢиҜҘж–№жі•иҝ”еӣһдәҶдёҖдёӘиҝӯд»ЈеҷЁеҜ№иұЎпјҢиҜҘиҝӯд»ЈеҷЁеҜ№иұЎдҪҝз”Ё yield е…ій”®еӯ—иҝ”еӣһдәҶдёүдёӘеҖј 1гҖҒ2гҖҒ3гҖӮз„¶еҗҺжҲ‘д»¬дҪҝз”Ё for...of еҫӘзҺҜйҒҚеҺҶ myIterable еҜ№иұЎпјҢжҜҸж¬Ўиҝӯд»ЈйғҪдјҡиҫ“еҮәеҪ“еүҚзҡ„еҖјгҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢеҸҜиҝӯд»ЈеҜ№иұЎе№¶дёҚдёҖе®ҡжҳҜж•°з»„жҲ–зұ»ж•°з»„еҜ№иұЎпјҢеҸӘиҰҒе®һзҺ°дәҶ Symbol.iterator ж–№жі•е№¶иҝ”еӣһдәҶиҝӯд»ЈеҷЁеҜ№иұЎпјҢйӮЈд№Ҳе®ғе°ұжҳҜдёҖдёӘеҸҜиҝӯд»ЈеҜ№иұЎгҖӮйҖҡиҝҮ for...of еҫӘзҺҜйҒҚеҺҶеҸҜиҝӯд»ЈеҜ№иұЎпјҢеҸҜд»ҘйҒҚеҺҶе®ғзҡ„жүҖжңүеҖјгҖӮ

Mapе’ҢSet вҖӢ

Map е’Ң Set жҳҜ ES6 дёӯж–°еўһзҡ„дёӨдёӘж•°жҚ®з»“жһ„пјҢе®ғд»¬еҲҶеҲ«з”ЁдәҺеӯҳеӮЁй”®еҖјеҜ№е’Ңе”ҜдёҖеҖјгҖӮ

Map вҖӢ

Map жҳҜдёҖз§ҚеҸҜиҝӯд»Јзҡ„й”®еҖјеҜ№йӣҶеҗҲпјҢе…¶дёӯзҡ„жҜҸдёӘй”®е’ҢеҖјйғҪеҸҜд»ҘжҳҜд»»ж„Ҹзұ»еһӢзҡ„ж•°жҚ®гҖӮе®ғзұ»дјјдәҺеҜ№иұЎпјҢдҪҶжңүдёҖдәӣзү№зӮ№пјҡ

  • еҜ№иұЎзҡ„й”®еҸӘиғҪжҳҜеӯ—з¬ҰдёІжҲ– Symbol зұ»еһӢпјҢиҖҢ Map зҡ„й”®еҸҜд»ҘжҳҜд»»ж„Ҹзұ»еһӢгҖӮ
  • Map дёӯзҡ„й”®еҖјеҜ№жҳҜжңүеәҸзҡ„пјҢиҖҢеҜ№иұЎдёӯзҡ„й”®еҖјеҜ№жҳҜж— еәҸзҡ„гҖӮ

дёӢйқўжҳҜ Map зҡ„дёҖдәӣеёёз”Ёж“ҚдҪңпјҡ

еҲӣе»ә Map вҖӢ

еҸҜд»ҘйҖҡиҝҮд»ҘдёӢж–№ејҸеҲӣе»әдёҖдёӘ Mapпјҡ

javascriptCopy code
const map = new Map();
ж·»еҠ й”®еҖјеҜ№ вҖӢ

еҸҜд»ҘдҪҝз”Ё set() ж–№жі•еҗ‘ Map дёӯж·»еҠ й”®еҖјеҜ№пјҡ

pythonCopy code
map.set('key', 'value');
иҺ·еҸ–еҖј вҖӢ

еҸҜд»ҘдҪҝз”Ё get() ж–№жі•иҺ·еҸ–жҢҮе®ҡй”®зҡ„еҖјпјҡ

goCopy code
const value = map.get('key');
еҲӨж–ӯй”®жҳҜеҗҰеӯҳеңЁ вҖӢ

еҸҜд»ҘдҪҝз”Ё has() ж–№жі•еҲӨж–ӯжҢҮе®ҡй”®жҳҜеҗҰеӯҳеңЁпјҡ

goCopy code
const hasKey = map.has('key');
еҲ йҷӨй”®еҖјеҜ№ вҖӢ

еҸҜд»ҘдҪҝз”Ё delete() ж–№жі•еҲ йҷӨжҢҮе®ҡй”®зҡ„й”®еҖјеҜ№пјҡ

goCopy code
map.delete('key');
жё…з©ә Map вҖӢ

еҸҜд»ҘдҪҝз”Ё clear() ж–№жі•жё…з©ә Map дёӯзҡ„жүҖжңүй”®еҖјеҜ№пјҡ

pythonCopy code
map.clear();
йҒҚеҺҶ Map вҖӢ

еҸҜд»ҘдҪҝз”Ё for...of еҫӘзҺҜйҒҚеҺҶ Map дёӯзҡ„жүҖжңүй”®еҖјеҜ№пјҡ

cCopy codefor (const [key, value] of map) {
  console.log(key, value);
}
иҺ·еҸ– Map зҡ„й•ҝеәҰ вҖӢ

еҸҜд»ҘдҪҝз”Ё size еұһжҖ§иҺ·еҸ– Map дёӯй”®еҖјеҜ№зҡ„ж•°йҮҸпјҡ

js
goCopy code
const length = map.size;
