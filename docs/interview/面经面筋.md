## 面经面筋

1. 请解释一下什么是JavaScript的作用域？

   ------

   JavaScript 中的作用域指的是在代码中*变量和函数*的可访问范围。它决定了在何处可以访问特定变量和函数。

   在 JavaScript 中，有全局作用域和局部作用域两种。全局作用域指的是在整个代码文件内可以访问的变量和函数，而局部作用域指的是仅在函数内部可以访问的变量和函数。

   全局变量在任何位置都可以访问，而局部变量只能在其定义的函数内访问。通过使用不同的作用域，您可以防止变量的命名冲突，并组织代码。

   

2. 请解释一下闭包的概念？

   ------

   闭包是指一个函数与其相关联的词法环境（该环境包含了该函数定义时存在的所有变量和函数）组合而成的对象。

   在 JavaScript 中，一个函数可以访问其定义时存在的所有变量和函数，即使该函数在其定义的上下文外被调用。通过创建闭包，您可以创建可以访问局部变量的函数，即使该函数已经离开了其定义的作用域。

   闭包可以用于维护函数内部的状态，并且可以捕获函数外部的变量。这使得闭包非常有用，因为它们可以在不修改外部状态的情况下保持内部状态。

   

3. 请说出三种创建JavaScript对象的方式？

   ------

   1. 使用 `new` 关键字创建对象：通过使用 `new` 关键字创建对象，您可以使用构造函数创建一个新的对象实例。这种方法的工作原理是，创建一个新的空对象，然后将该对象的原型链指向构造函数的原型，最后调用构造函数，并将新创建的对象作为 `this` 传递给构造函数。

   2. 使用构造函数创建对象：构造函数是一种特殊类型的函数，它用于创建对象。您可以通过调用构造函数并使用 `new` 关键字创建对象。

   3. 使用原型方法创建对象：原型方法是一种在 JavaScript 中创建对象的方法。该方法使用一个空对象作为原型，并向该对象添加属性和方法。然后，您可以通过创建新对象并将其原型链指向该对象来创建新的对象实例。

5. 请说出两种使用JavaScript实现继承的方式？

   ------

   JavaScript 中有多种实现继承的方式，其中两种常用的方法是原型继承和构造函数继承。

   原型继承是利用 JavaScript 原型链的思想，将一个对象的原型指向另一个对象，从而实现继承。代码如下：

   ```js
   javascriptCopy codefunction Parent() {}
   Parent.prototype.foo = function() {
     console.log('foo');
   };
   
   function Child() {}
   Child.prototype = new Parent();
   
   var child = new Child();
   child.foo(); // 输出 'foo'
   ```

   构造函数继承是在子类的构造函数中调用父类的构造函数，从而实现继承。代码如下：

   ```js
   javascriptCopy codefunction Parent() {}
   Parent.prototype.foo = function() {
     console.log('foo');
   };
   
   function Child() {
     Parent.call(this);
   }
   
   var child = new Child();
   child.foo(); // 抛出错误，因为 Child 对象没有 foo 方法
   ```

   需要注意的是，构造函数继承无法实现继承原型上的方法，所以如果需要实现原型继承的功能，需要结合原型继承的方法使用。
