# PEGA

### 什么是PEGA?

​	基于java和java oop的业务流程管理（BPM）工具。主要功能是ega的主要作用是帮助企业优化和自动化各种业务流程，并通过集成不同的系统和数据源来提高业务效率和客户满意度。

​	可以做什么？

1. **业务流程优化和自动化**
2. **客户关系管理**
3. **决策管理**
4. **自定义应用开发**
